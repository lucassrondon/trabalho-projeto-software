
# Lista de Compras

Este é um aplicativo web desenvolvido com React, Vite e Tailwind CSS. Ele permite que usuários adicionem itens à lista de compras, calculem o total em tempo real, finalizem compras e visualizem o histórico de compras. A aplicação também possui suporte a PWA, permitindo instalação em dispositivos.

---

## **Recursos**
- Adicionar e remover itens da lista de compras.
- Visualizar o total gasto em tempo real.
- Finalizar a compra e salvar o histórico.
- Histórico de compras com detalhes de itens e totais.
- Suporte a Progressive Web App (PWA).

---

## **Pré-requisitos**
Certifique-se de ter instalado:
- **Node.js** (versão 16 ou superior)
- **npm** ou **yarn**

---

## **Como Rodar a Aplicação**

### 1. Clone o Repositório
```bash
git clone <URL_DO_REPOSITORIO>
cd lista-de-compras
```

### 2. Instale as Dependências
Execute o comando:
```bash
npm install
```

### 3. Rode o Servidor de Desenvolvimento
Para iniciar o servidor local:
```bash
npm run dev
```

Acesse a aplicação em [http://localhost:5173](http://localhost:5173).

---

## **Build para Produção**

Para gerar os arquivos otimizados para produção:
```bash
npm run build
```

Para pré-visualizar o build:
```bash
npm run preview
```

---

## **Progressive Web App (PWA)**

1. Acesse a aplicação no navegador.
2. Clique no ícone de "instalar" (ou vá até as opções do navegador e selecione "Adicionar à tela inicial").
3. Aproveite a aplicação como um app nativo!
